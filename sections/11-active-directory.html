Forest & Domain Structure
  • ForestThe true security boundary. Shared schema, GC, and Enterprise Admins group.
  • DomainAdmin and replication boundary within a forest. Has its own DC, krbtgt, and domain admins.
  • OUOrganizational Unit. Container for GPO application and delegated admin. Not a security boundary.
  • GCGlobal Catalog. Partial replica of all objects in the forest.
  • FSMO Roles5 single-master roles: Schema Master, Domain Naming Master (forest); PDC Emulator, RID Master, Infrastructure Master (per domain).

SID Filtering
Cross-forest trusts apply SID Filtering by default — strips SIDs from foreign domains out of the PAC at the trust boundary. Within a single forest, SID history is trusted.
Group Scopes
  • Domain LocalMembers from any domain. Used for resource permissions in same domain.
  • GlobalMembers from same domain only. Usable in any domain. Replicated to own DC only.
  • UniversalMembers from any domain in forest. Replicated to Global Catalog.

AD Object Security Descriptors

Every AD object has a full security descriptor stored in the directory. ACEs can apply to the object itself or be scoped to specific object types (classes) or properties (attributes).

  • Standard Access RightsGenericAll, GenericRead, GenericWrite, GenericExecute — mapped to AD-specific rights.
  • Object-Specific RightsDS-Read-Property, DS-Write-Property scoped to specific attribute GUIDs.
  • Control Access RightsExtended rights identified by GUID. Examples: Reset-Password, DS-Replication-Get-Changes-All.
  • Property SetsGroups of attributes under one right GUID.
  • Write-Validated RightsConstrain writes to validated values only (e.g., Self-Membership).
  • AdminSDHolderTemplate object whose ACL is propagated to all protected group members by SDProp every 60 minutes.

Security-Relevant GPO Settings

  • Password PolicyMin length, complexity, history, max age. One per domain unless PSO used.
  • Audit PolicyCategories logged. Set at domain or OU level.
  • User RightsURA / privilege assignments per machine.
  • Restricted GroupsEnforce local group membership on machines.
